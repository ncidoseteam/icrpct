# ICRP CT Dose Calculator JavaScript Version

This folder contains the standalone JavaScript/HTML version and the data files it needs.

Run it with a local web server from this folder:

```sh
python3 -m http.server 8000
```

Then open:

```text
http://127.0.0.1:8000/icrpct_dose_calculator.html
```

Do not open the HTML directly with `file://`, because browsers block loading the CSV files that way.
